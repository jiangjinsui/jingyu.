☁️ GitHub 云端备份教程

原理说明

静语会把你的字卡库和聊天数据加密存到你的 GitHub 私人仓库里，完全只有你能看到，免费且安全。

第一步 · 创建 GitHub 账号

如果还没有 GitHub 账号，前往 github.com 免费注册一个。

第二步 · 创建私密仓库

  1. 登录 GitHub，点击右上角 + → New repository
  2. Repository name 填写：jingyu-backup（或任意名字）
  3. 选择 Private（重要！保证私密）
  4. 勾选 Add a README file
  5. 点击 Create repository

创建后页面地址类似：github.com/你的用户名/jingyu-backup

第三步 · 生成访问 Token

  1. 点击右上角头像 → Settings
  2. 左侧最底部找到 Developer settings
  3. 点击 Personal access tokens → Tokens (classic)
  4. 点击 Generate new token (classic)
  5. Note 随便填，如：静语备份
  6. Expiration 建议选 No expiration
  7. 勾选权限：repo（第一个大选项，全勾上）
  8. 拉到最下面点 Generate token
  9. ⚠️ 页面出现 ghp_xxxxxx... 这串就是你的 Token，立即复制，关闭页面就再也看不到了

第四步 · 填入静语

  1. 回到静语设置页面的「GitHub 云端备份」区域
  2. 在 GitHub Token 框粘贴刚复制的 Token
  3. 在 用户名/仓库名 框填写，如：alice/jingyu-backup
  4. 设置自动备份间隔（如 24 小时）
  5. 勾选「自动备份」
  6. 点击「保存设置」，再点「立即备份」测试一下
  7. 看到 ✅ 备份成功即大功告成！

恢复备份

在 GitHub 仓库里找到「静语备份.json」文件，点击下载，然后用设置里的「一键导入备份」导回来即可。

⚠️ Token 很重要，不要分享给任何人。如果泄露，立刻去 GitHub 删掉该 Token 再重新生成。
